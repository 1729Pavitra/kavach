# Kavach

**kavach** is a minimal shell written out of pure curiousity. Hopefully, **kavach** would help someone who wants to understand how shells work.

## Installation

```
$ git clone http://github.com/anayjoshi/kavach
$ cd kavach
$ make
$ ./kavach
```

To install **kavach**, execute

```
$ sudo make install
```

To uninstall **kavach**:

```
$ sudo make uninstall
```
